# ndoc

Попытка небольшой автоматизации подачи ежемесячных деклараций на rs.ge с помощью puppeteer
Работает только для Liberty

## Установка

```shell
sudo npm i -g yarn
yarn
yarn css
```

## Запуск

```shell
yarn start
```

## Создание бинарника

```shell
yarn package
```

Результат появится в директории `./out`
